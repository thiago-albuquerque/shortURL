# shortURL

Short URL é um App Mobile simples para treinar requisições em API. 
Nele o usuário cola ou digita uma URL e um nome e em seguida gera uma outra URL mais curta e personalizada.

## Tecnologias utilizadas:

- Javascript
- Styled Component
- React Native
- Hook
- Clipboard
- API da 'Cuttly"
